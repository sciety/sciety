## Technical Stack

- **Protocols**: ActivityPub, COAR Notify (LDN + ActivityStreams2), DocMaps (JSON-LD)
- **Identity**: ORCID OAuth
- **Archival**: Zenodo REST API
- **Federation**: Mastodon, Bluesky (via Bridgy Fed), Fediverse
- **Standards**: W3C ActivityPub, COAR Notify, DocMaps Framework
