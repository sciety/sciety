# 3d. Research and design UI in Sciety for the integration

## Additional design activity for the Bonfire Networks UI

- Provides design for UI that could enable Sciety users to post public “discussion” of preprints they discover on Sciety.
- Provides design for UI that could enable Sciety users to discuss preprints privately with other Fediverse users.
- Slides contain design suggestions that can be treated as separate ideas, or a series of complimentary
releases.

Full set of slides representing this design work is located here: [Sciety x Bonfire](https://docs.google.com/presentation/d/1NI0CwOvnSW7PtgWRa8Mi3fXYa8JMqn7WMEtdbCufpvo/edit?usp=sharing)
